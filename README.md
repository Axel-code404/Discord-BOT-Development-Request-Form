# 🚀 Discord BOT開発依頼プラットフォーム

チャット形式で気軽にDiscord BOT開発を開発者へ依頼できるWebサービスです。

---

## 🌟 概要

本サービスは、

- Discord BOT開発を依頼したい人
- BOTを開発できるエンジニア

をつなぐためのWebプラットフォームです。

チャット形式で依頼内容を送信でき、開発者とのやり取りをスムーズに行うことができます。
